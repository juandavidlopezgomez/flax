Politica de Privacidad

Ultima actualizacion: 2026-04-21

1. Informacion que Recopilamos

La aplicacion "flax" utiliza autenticacion OAuth de TikTok y Kick. Accedemos unicamente a la informacion necesaria para publicar contenido: token de acceso, ID de usuario y permisos de subida de video.

2. Almacenamiento de Datos

Los tokens de acceso se almacenan localmente en el dispositivo del usuario. No enviamos datos a servidores externos. No recopilamos datos personales adicionales.

3. Uso de Datos

Los datos se usan exclusivamente para:

  • Autenticar al usuario con las plataformas
  • Descargar clips autorizados desde Kick
  • Subir clips a la cuenta TikTok del usuario

4. Terceros

La aplicacion interactua con las APIs oficiales de:

  • TikTok (Content Posting API)
  • Kick Developer API

Consulta las politicas de privacidad de cada plataforma para entender como manejan tus datos.

5. Seguridad

Los tokens se almacenan localmente. El usuario es responsable de proteger su dispositivo y credenciales.

6. Derechos del Usuario

Puedes revocar el acceso en cualquier momento desde la configuracion de tu cuenta TikTok o Kick. Tambien puedes eliminar los archivos locales de la aplicacion.

7. Menores de Edad

La aplicacion no esta dirigida a menores de 13 anos.

8. Cambios a esta Politica

Podemos actualizar esta politica. Los cambios se publicaran en esta misma URL.

9. Contacto

Para consultas: contactar al desarrollador a traves del repositorio GitHub.
